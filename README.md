## 4. 인플루언서 시스템

> **핵심 차별점** — 인스타에서 운동 기록하던 사람들을 달료 안으로 흡수.
> GPS·심박수 데이터가 붙은 오운완 포스팅은 신뢰도가 높아 브랜드 광고 단가도 올라감.
> 달료가 인플루언서 ↔ 브랜드 중간에서 수수료 수취.

### 오운완 피드 (예시)

```
러닝요정_지은 [인플루언서]
오늘 10.2km · 5'32"/km · 심박 158bpm

오늘도 달료! 한강 야경 코스 완주했어요.
달료가 페이스 잡아줘서 처음으로 네거티브 스플릿 성공 🏃‍♀️
#오운완 #달료런

♥ 248   ✉ 31   📊 GPS 기록 보기
```

### 인플루언서 등급

| 등급 | 기준 |
|------|------|
| Nano | 팔로워 1K~10K |
| Micro | 팔로워 10K~100K |
| Macro | 팔로워 100K~1M |
| Mega | 팔로워 1M+ |

### 브랜드 매칭

- 달료가 스포츠 브랜드 ↔ 인플루언서 중개
- 수수료 **15%** 수취
- GPS 인증 데이터로 광고 신뢰도 차별화

## 6. 캐릭터 성장 시스템

### 성장 공식

```
달린 거리 (km)  ×  가속 배율 (아이템)  =  경험치 (EXP)
```

- 기본: **1km = 10 EXP**
- 아이템 적용 시 최대 **3배**
- 연속 달리기 보너스 최대 **1.5배**

### 성장 단계

| 단계 | 조건 | 특징 |
|------|------|------|
| 🥚 알 | 가입 시 지급 (랜덤) | 기본 캐릭터 중 랜덤 부화 |
| 🐣 새싹 | 누적 10km | 첫 런 완료 후 부화 |
| 🐥 아기 | 누적 50km | 외형 변화 + 마이룸 아이템 해금 |
| 🏃 러너 | 누적 200km | 특수 모션 해금 |
| ⚡ 챔피언 | 누적 500km+ | 희귀 스킨 해금 |
| 🪨 돌 변신 | **7일 미달리기** | 방치 페널티 |

### 방치 페널티

```
3일차 → "달료가 뻣뻣해졌어요..."
5일차 → "달료 몸이 회색이 되고 있어요…"
7일차 → 완전 돌 변신 🪨
뛰면  → 즉시 해동 시작!
```

### 뽑기 시스템

| 등급 | 확률 |
|------|------|
| 일반 | 60% |
| 레어 | 25% |
| 에픽 | 12% |
| 전설 ✨ | 3% |

- 가입 시 알 1개 지급 → 첫 런 완료 시 랜덤 부화
- 시즌별 한정 캐릭터로 희소성 부여

### 마이룸

- 동물 친구 수집 (고양이, 햄스터, 토끼 등 — 뛸수록 해금)
- 벽지·가구·소품 포인트로 구매
- 팔로워 마이룸 방문 + 응원 메시지

## 7. 성장 가속 아이템샵

> 뛰는 것만으로도 성장하지만, 아이템으로 속도를 높일 수 있음.
> 강요 없이 "더 빨리 키우고 싶으면" 선택지로 제공.

### 아이템 목록

| 아이템 | 효과 | 가격 |
|--------|------|------|
| ⚡ 터보 부스터 | EXP 2배 · 24시간 | ₩990 |
| 🌟 주간 패스 | EXP 3배 · 7일 | ₩4,900 |
| 🛡️ 방치 방어막 | 돌 변신 방지 | ₩1,500 |
| 💧 해동 물약 | 돌 변신 즉시 복구 | ₩500 |
| 🎯 콤보 부스터 | 연속 달리기 보너스 배율 추가 증가 | ₩1,200 |
| 🎁 월간 성장팩 | EXP 2배 + 방어막 + 해동 물약 2개 | ₩7,900 |

### 포인트 시스템

- 달릴수록 포인트(무료 재화) 적립
- 포인트로 일부 아이템 구매 가능
- 과금 없이 아이템 경험 유도 → 유료 전환율 증가
