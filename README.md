## 2. 시장 분석

### 경쟁사 포지셔닝

| 앱 | 강점 | 약점 |
|----|------|------|
| Nike Run Club | 무료, 코칭 콘텐츠, 월 40만 다운로드 | AI 개인화 없음, 캐릭터/감성 없음 |
| Strava | 소셜 커뮤니티 | 딱딱한 UI, 코칭 기능 부족 |
| Garmin Connect | 정밀 기기 데이터 | 기기 종속, 노후화된 UX |
| Zombies, Run! | 스토리 기반 게이미피케이션, 1천만 다운로드 | 어두운 세계관, 귀여운 감성 없음 |

### 시장 공백 (달료의 진입점)

> 귀여운 캐릭터 + AI 실시간 코칭 + GPS 인증 인플루언서 생태계 + **지역 사투리 감성**을 모두 갖춘 앱은 없음.

### 달료의 해자

- **GPS 인증 오운완** — 인스타 오운완은 자기 신고지만 달료는 실제 데이터가 붙음.
- **캐릭터 애착** — 돌 변신 페널티 + 성장 애착은 일반 배지 시스템보다 이탈 방어력이 높음.
- **카메라 필터 바이럴** — 필터 사진마다 달료 워터마크가 따라가는 무료 마케팅 구조.
- **사투리 감성** — 경쟁 앱 전무. 지역 유저 소속감 + SNS 공유 유도 효과.

## 4. 사투리 시스템

### 개요

앱 설정에서 지역 사투리를 선택하면 달료의 모든 말투가 해당 사투리로 바뀜.
버튼 텍스트, 운동 안내문, 푸시 알림, 캐릭터 반응 대사 등 전체 적용.

### 지원 방언

| 지역 | 코드 |
|------|------|
| 표준어 (기본) | `ko-std` |
| 경상도 | `ko-gyeong` |
| 전라도 | `ko-jeon` |
| 충청도 | `ko-chung` |
| 제주도 | `ko-jeju` |

### 적용 범위

- 버튼 텍스트 (달료!, 멈쵸! 등)
- 운동 중 안내 메시지
- 운동 완료 화면 문구
- 캐릭터 리액션 대사
- 푸시 알림 텍스트
- 카메라 필터 오버레이 문구 (선택)
- 자유게시판 기본 해시태그 (예: #경상도달료런)

### 설정 위치

- 앱 설정 > 달료 말투 > 지역 선택
- 온보딩 3단계에서 미리 선택 권유

### 바이럴 포인트

카메라 필터 사진을 인스타에 공유할 때 사투리 문구가 포함되면 **지역 커뮤니티 공유 확률이 높아짐**.
"경상도 달료런" "전라도 달료런" 같은 지역 해시태그가 자연스럽게 생성될 수 있음.

## 5. 인플루언서 시스템

> **핵심 차별점** — 인스타에서 운동 기록하던 사람들을 달료 안으로 흡수.
> GPS·심박수 데이터가 붙은 오운완 포스팅은 신뢰도가 높아 브랜드 광고 단가가 올라감.
> 달료가 인플루언서 ↔ 브랜드 중간에서 수수료 수취.

### 왜 MVP에 인플루언서 기능이 있어야 하는가

운동 인스타 유저가 달료에 계정을 만드는 이유는 "여기서 인플루언서가 될 수 있어서"여야 한다.
그 근거가 처음부터 없으면 GPS 기록 앱과 다를 게 없고, 초기 유입 전략 자체가 무너진다.

### 달료 카메라 필터

```
[ 내 달료 캐릭터 ] + [ 오늘 10.2km · 5'32"/km · 심박 158bpm ]
                    ↓
           AR 필터로 사진 촬영 (사투리 문구 포함 선택 가능)
                    ↓
     달료 워터마크 자동 삽입 후 인스타·카카오 공유
```

### 오운완 피드 (예시)

```
러닝요정_지은 [인플루언서 · Micro]
오늘 10.2km · 5'32"/km · 심박 158bpm

오늘도 달료! 한강 야경 코스 완주했어요.
달료가 페이스 잡아줘서 처음으로 네거티브 스플릿 성공 🏃‍♀️
#오운완 #달료런

♥ 248   ✉ 31   📊 GPS 기록 보기
```

### 인플루언서 등급

| 등급 | 팔로워 기준 | 혜택 |
|------|------------|------|
| Nano | 1K~10K | 인플루언서 배지, 기본 분석 대시보드 |
| Micro | 10K~100K | 브랜드 매칭 신청 가능, 고급 대시보드 |
| Macro | 100K~1M | 우선 브랜드 매칭, 전용 지원 |
| Mega | 1M+ | 파트너십 계약, 공동 캠페인 |

### 브랜드 매칭

- 달료가 스포츠 브랜드 ↔ 인플루언서 중개
- 수수료 **15%** 수취
- GPS 인증 데이터로 광고 신뢰도 차별화

## 7. 캐릭터 성장 시스템

### 성장 공식

```
달린 거리 (km)  ×  가속 배율 (아이템)  =  경험치 (EXP)
```

- 기본: **1km = 10 EXP**
- 아이템 적용 시 최대 **3배**
- 연속 달리기 보너스 최대 **1.5배**

### 성장 단계

| 단계 | 조건 | 특징 |
|------|------|------|
| 🥚 알 | 가입 시 지급 (랜덤) | 기본 캐릭터 중 랜덤 부화 |
| 🐣 새싹 | 누적 10km | 첫 런 완료 후 부화 |
| 🐥 아기 | 누적 50km | 외형 변화 + 마이룸 아이템 해금 |
| 🏃 러너 | 누적 200km | 특수 모션 해금 + 필터 이펙트 강화 |
| ⚡ 챔피언 | 누적 500km+ | 희귀 스킨 해금 + 최고급 필터 이펙트 |
| 🪨 돌 변신 | **7일 미달리기** | 방치 페널티 |

### 방치 페널티 (표준어 / 경상도 예시)

```
3일차 → "달료가 뻣뻣해졌어요..." / "달료가 굳어뿌고 있다 아이가"
5일차 → "달료 몸이 회색이 되고 있어요…" / "달료 몸이 회색 됐다 아이가…"
7일차 → 완전 돌 변신 🪨
뛰면  → 즉시 해동 시작!
```

### 뽑기 시스템

| 등급 | 확률 |
|------|------|
| 일반 | 60% |
| 레어 | 25% |
| 에픽 | 12% |
| 전설 ✨ | 3% |

### 마이룸

- 동물 친구 수집 (고양이, 햄스터, 토끼 등 — 뛸수록 해금)
- 벽지·가구·소품 포인트로 구매
- 팔로워 마이룸 방문 + 응원 메시지

## 10. MVP 범위

MVP는 두 가지 핵심 루프를 완성하는 데 집중한다.
1. **달리면 캐릭터가 자란다** — 재방문과 이탈 방어
2. **여기서 인플루언서가 될 수 있다** — 초기 유입 동기

### MVP 포함 (9개)

| 기능 | 비고 |
|------|------|
| 달료 캐릭터 & 속도 모션 | |
| km 기반 성장 & 돌 변신 페널티 | |
| 알 온보딩 (랜덤 부화) | |
| GPS 기록 & 오운완 피드 | |
| 달료 카메라 필터 & 외부 공유 | |
| 인플루언서 등급 시스템 | 2단계 → MVP 상향 |
| 팔로우 & 피드 구독 | 2단계 → MVP 상향 |
| 달료 워터마크 자동 삽입 | |
| **사투리 시스템** | ✨ 신규 |

### 2단계 (3~6개월)

| 기능 | 미루는 이유 |
|------|------------|
| AI 페이스 코치 | API 비용은 유저 풀 확보 후 관리 |
| 성장 가속 아이템샵 | 캐릭터 애착 생긴 후가 효과적 |
| 랜덤 뽑기 (가챠) | 확률형 아이템 규제 법무 검토 필요 |
| 러닝크루 | 오운완 피드 정착 후 연결 |
| 자유게시판 | 커뮤니티 밀도 쌓인 후 개설 |
| 마이룸 & 포인트샵 | 캐릭터 애착 지표 확인 후 투자 결정 |

### 3단계 이후 (6개월~)

| 기능 | 조건 |
|------|------|
